# Roadmap

## Phase 0 — Foundation ✅
- 프로젝트 스캐폴드
- PRD/아키텍처/로드맵 문서
- API 헬스/상태 엔드포인트

## Phase 1 — Core MVP ✅
- Google OAuth 연결 API 골격
- Workspace 액션 실행 API
- GitHub Webhook 이벤트 저널
- 일/주간 보고서 생성기
- 문서베이스 CRUD + 검색
- 팀 채팅 채널/메시지
- 세금계산서 Draft/Issue 워크플로
- 자연어 Orchestrator `/agent/execute`

## Phase 2 — Real Integrations (다음)
- Google 실제 API 토큰 교환/권한 적용
- GitHub App 설치형 인증 + 저장소 범위 제어
- DeepAgents SDK 직접 연결 + subagent streaming
- 문서베이스 벡터 검색/참조 근거 추적

## Phase 3 — Collaboration Hub
- 실시간 채팅(WebSocket)
- 프로젝트 보드(칸반/스프린트)
- 위키/정책/회의록 템플릿 운영

## Phase 4 — Billing Automation
- Barobill 실제 발행 API 연동
- 승인 플로우 + 회계 연계
- 월간 자동 정산 리포트
