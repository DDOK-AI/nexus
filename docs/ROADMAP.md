# Roadmap

## Phase 0 — Foundation (현재)
- 프로젝트 스캐폴드
- PRD/아키텍처/로드맵 문서
- API 헬스/상태 엔드포인트

## Phase 1 — Workspace + GitHub Core
- Google OAuth 및 Workspace API 커넥터
- GitHub App 연동
- DeepAgents orchestrator 기본 라우팅

## Phase 2 — Reporting + Knowledge Base
- 일일 R&D 보고서 자동 생성
- 주간보고 템플릿화 및 자동 배포
- 프로젝트 문서베이스(검색/요약) 구축

## Phase 3 — Collaboration Hub
- 팀 채팅
- 작업보드/문서/이슈 통합 뷰
- 에이전트에게 문서베이스 권한 기반 접근 제공

## Phase 4 — Billing Automation (Future)
- 바로빌 API 연동
- 세금계산서 발행 자동화
- 회계 시스템 연계
